# RLongdoDict
Longdo Dictionary in RStudio

###Install RLongdoDict
```{r install_devtools, eval=FALSE}
library("devtools")
install_github("slphyx/RLongdoDict")
```

###Using RLongdoDict
```{r example}
LongdoDict("tutelage")
```
